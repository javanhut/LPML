LPML

The Lazy Page Maker Language

Build beautiful web pages without touching HTML or CSS

Why LPML?

Simple Syntax

Write human-readable markup. No angle brackets, no closing tag confusion.

Easy Styling

Use words like 'large' and 'medium' instead of memorizing pixel values.

Fast Output

Compiles to clean HTML instantly. Works in any browser.

See It In Action

LPML Input

[p-start]
  contains = "Hello World!"
  color = "blue"
  text_size = "large"
  format_with = ["bold"]
[p-end]

HTML Output

<p style="color: blue;
   font-size: 24px;">
  <strong>Hello World!</strong>
</p>

Quick Start

1. Install

git clone https://github.com/yourusername/lpml
cd lpml
go build -o lpml .

2. Create a file

[mid-page-start]
  [h-start]
    contains = "My First LPML Page"
    color = "navy"
  [h-end]
  [p-start]
    contains = "This is so easy!"
  [p-end]
[mid-page-end]

3. Compile

./lpml mypage.lpml
# Opens mypage.html in browser
